What is Pinax?

Pinax is an open-source platform based on Django, intended to provide a starting point for websites. It takes care of the things that many sites have in common, so you can focus on what makes your site different.


Starter projects providing scaffolding, already integrated components and/or sample code.

Reusable apps providing both back-end functionality and user-facing components.

Default templates enabling quick Bootstrap-based prototyping and easy customization.

See pinaxproject.com for more information.
